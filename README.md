# 🛍️ Customer Purchase Analysis

An end-to-end Data Analytics project that transforms raw customer shopping data into actionable business insights using **Python**, **MySQL**, and **Power BI**.

The project demonstrates the complete analytics lifecycle:
- Data Cleaning & Feature Engineering using Python
- Business Analysis using SQL
- Interactive Dashboard Development using Power BI

---

# 📌 Business Problem

Retail businesses generate thousands of customer transactions every day. The objective of this project is to analyze customer purchasing behavior, subscription patterns, discounts, product performance, and revenue trends to support data-driven business decisions.

# 🐍 Phase 1 – Python

## Data Preparation

### Dataset Inspection

- Loaded customer shopping dataset
- Explored dataset structure
- Generated descriptive statistics
- Identified missing values

### Data Cleaning

- Filled missing Review Ratings using category-wise median
- Renamed columns into SQL-friendly format
- Standardized column names
- Removed special characters from column names

### Feature Engineering

Created additional business columns:

- Age Group
  - Young Adult
  - Adult
  - Middle-aged
  - Senior

- Purchase Frequency (Days)
  - Weekly → 7
  - Fortnightly → 14
  - Monthly → 30
  - Quarterly → 90
  - Annually → 365

### Exploratory Data Analysis

- Dataset Summary
- Missing Value Analysis
- Statistical Analysis
- Customer Demographics
- Purchase Behaviour Analysis
